# YouYou - 本地智能助手系统

一个完全本地运行的智能助手,支持自然语言对话、物品位置记忆等功能。采用多 Agent 架构,使用 LLM 进行智能路由。

## 特性

- 智能对话 - 自然语言交互,友好的对话体验
- 物品记忆 - 记录和查询物品位置,支持语义检索
- 多 Agent 架构 - 清晰的职责划分,易于扩展
- LLM 路由 - 自动识别用户意图,路由到对应功能
- 本地运行 - 数据完全本地存储,保护隐私
- 美观界面 - Rich 终端美化,提升交互体验

## 快速开始

### 1. 环境要求

- Python 3.11+
- uv 包管理器

### 2. 安装

```bash
# 使用 uv 同步依赖
uv sync
```

### 3. 配置

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件,填写你的 OpenAI API 配置
```

### 4. 运行

```bash
# 启动 YouYou 助手
uv run youyou
```

## 使用示例

### 物品位置记忆

```
你: 钥匙在客厅桌上
YouYou: 好的,我已经记住了,钥匙在客厅桌上。

你: 钥匙在哪?
YouYou: 钥匙在客厅桌上。
```

### 普通对话

```
你: 你好
YouYou: 你好!我是 YouYou,你的个人助理。
```
